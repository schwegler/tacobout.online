🎨 Palette: Add ALT text badges for images

What: Added a new JavaScript file and CSS styles to automatically wrap images that have alt text with a visual "ALT" badge. Clicking the badge reveals the alt text in a tooltip.
Why: This improves accessibility and UX by making alt text easily discoverable for users who are not using screen readers.
Before/After: Images now have a small "ALT" badge in the bottom right corner if they have an `alt` attribute. Clicking the badge reveals a tooltip with the `alt` text.
Accessibility: The badge is fully keyboard accessible, using a `<button>` element with `aria-label` and `focus-visible` states. The tooltip uses `aria-hidden` attributes to hide it from screen readers when not active, and the JS updates the attributes appropriately.
